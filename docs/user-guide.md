# 用户指南

## 成绩档案

启动程序后，可以新建成绩档案或打开已有 `.db` 文件。一个档案可保存多个学生，以学号作为唯一标识。请自行保存和备份 `.db` 文件。

## 录入课程

课程记录包含学期、课程编号（可空）、课程名称、整数成绩、学分和是否计入计算。学期格式为 `X-(X+1)-[1-3]`，例如 `2025-2026-2`。来源会显示为“导入”“修改”或“手动”；计入列用绿色圆点表示参与计算，灰色圆点表示不参与计算。

## 导入北邮教务成绩

1. 打开北邮教务系统的“课程成绩”页面并查询成绩；
2. 在成绩表格内部点击一次，按 `Ctrl+A` 全选表格，再按 `Ctrl+C` 复制完整表格（包含表头）；
3. 在程序的“课程记录”页选择“导入教务成绩”；
4. 在导入窗口中点击“识别成绩表”；如果文本框为空，程序会读取剪贴板并解析；也可以先手动粘贴再识别；
5. 核对预览后确认导入。

程序按表头识别开课学期、课程编号、课程名称、成绩和学分，不依赖固定列顺序。重复导入时，会按“学期 + 课程编号”更新已有课程，并重建为本次教务表的课程顺序；无编号课程仅在同学期、同名且唯一时合并。它不会登录教务系统，也不会保存登录信息。

## 计算范围

在“成绩概览”右上角可以选择全部学期、某一学年或单个学期。仅标记为“计入”的课程参与 GA 和 GPA；0 学分课程保留在列表中，但不会改变加权结果。GA 与 GPA 显示四位小数，学分显示一位小数；第四张卡片固定显示最新学期的 GA，并在选择学年或单学期时变灰，避免和当前范围混淆。

## 排序与草稿

点击课程表任一列标题会依次切换升序、降序、默认顺序。默认顺序按学期排列，同一学期内先显示教务导入课程（教务表顺序），再显示手动课程。新增和编辑课程均通过弹层完成，课程表本身只负责展示与选择；编辑导入课程后，其来源会变为“修改”。

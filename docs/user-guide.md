# 用户指南

## 成绩档案

启动程序后，可以新建成绩档案或打开已有 `.db` 文件。一个档案可保存多个学生，以学号作为唯一标识。请自行保存和备份 `.db` 文件。

## 录入课程

课程记录包含学期、课程编号（可空）、课程名称、整数成绩、学分和是否计入计算。学期格式为 `X-(X+1)-[1-3]`，例如 `2025-2026-2`。

## 导入北邮教务成绩

1. 打开北邮教务系统的“课程成绩”页面并查询成绩；
2. 点击结果表，按 `Ctrl+A`、`Ctrl+C`；
3. 在程序的“课程记录”页选择“导入教务成绩”；
4. 在输入框中按 `Ctrl+V`，核对预览后确认导入。

程序按表头识别开课学期、课程编号、课程名称、成绩和学分，不依赖固定列顺序。它不会登录教务系统，也不会保存登录信息。

## 计算范围

可以选择全部学期、单学期或多个学期。仅标记为“计入计算”的课程参与 GA 和 GPA；0 学分课程保留在列表中，但不会改变加权结果。

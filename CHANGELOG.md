# 更新日志

本文件记录 BUPT GPA Calculator 面向用户的重要变化。

## Unreleased

### Fixed

- 对齐成绩概览页上下卡片间距，使图表卡片与上方指标卡片边缘一致。

### Known Issues

- 新增学生并自动切换到新学生前，没有提示保存当前学生的未保存课程，可能导致未保存数据丢失。见 [#6](https://github.com/lte-z/BUPT-GPA-Calculator/issues/6)。
- 打开包含较多数据的 `.db` 成绩档案时，界面可能短暂未响应，需要优化档案读取与界面刷新流程。见 [#7](https://github.com/lte-z/BUPT-GPA-Calculator/issues/7)。

## v0.1.0 - 2026-07-14

- 初始化 WPF、核心逻辑与测试项目骨架；
- 建立中文文档、Issue、PR、CI 与 Release 维护流程；
- 重构为 WPF-UI Fluent 界面，跟随 Windows 亮/暗主题与系统强调色；
- 教务成绩导入改为“识别成绩表”单按钮流程，并保存教务顺序、来源标签与增量合并结果；
- 课程来源改为导入、修改、手动三态，计入状态改为绿色/灰色圆点显示；
- 重建课程录入流程，课程表改为只读展示，新增、编辑、导入、确认与报错均使用 Fluent 弹层；
- 概览支持全部、学年、单学期三种范围，趋势图支持 GA/GPA 切换，第四张概览卡显示最新学期 GA；
- 增加窗口图标、使用说明页、完整用户指南入口与空列表引导；
- 修复空白手动课程可能导致界面异常的问题，草稿不再影响统计；
- 新增导入合并与真实表格结构的虚构数据测试。

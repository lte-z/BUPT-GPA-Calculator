# 安全与隐私说明

BUPT GPA Calculator 是本地运行的 Windows 程序。设计目标是不向网络发送用户的成绩档案，也不直接登录北邮教务系统。

## 敏感数据

请不要提交、公开或粘贴以下内容：

- `.db` 成绩档案及其 SQLite 旁文件；
- 教务系统复制的完整成绩表；
- 学号、姓名、本机绝对路径或截图中的个人信息；
- 账号、密码、Cookie、VPN 或其他认证信息。

## 数据边界

用户自行选择成绩档案的保存位置。程序只在用户打开的 `.db` 文件中保存数据，不使用 AppData、注册表或云端服务保存成绩。

## 报告问题

发现安全、隐私或数据损坏风险时，请在 GitHub Issue 中仅提供最小化、脱敏后的复现信息；不要上传真实成绩档案或认证数据。
